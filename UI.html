SIGN-IN PAGE

Introductory page of the website and the first thing a user sees when using the application.

SIGN-UP PAGE-1

When the user clicks, "Get Started" from the main page they are led to this page where they can either sign in with facebook or login as a "Guest" limited to viewing events only.

SIGN-UP PAGE-2

The first time a user logs in to LetsGo using Facebook, this page will display and prompt the user to their Facebook friends who are already on LetsGo. The user can also invite their friends who are not on the app.

EVENTS PAGE-1

Hovering over an event box shows a more detailed view of the event. The user may click the + button to save an event to their profile. The user will also be able to edge scroll the reel of events to the right.

EVENTS PAGE-2

Clicking on an event will display this pop up consisting of two tabs, one being the twitter feed tab, where you can see what people are tweeting about the event.

EVENTS PAGE-3

Another being the friend tab, where you can see a list of your friends that are going.

EVENTS PAGE-4

Clicking on the fixed search bar at the top of the screen will allow the user to search for a particular event.

EVENTS PAGE-5

Clicking on the profile button will drop down the options available to the User.

EVENTS PAGE-6

Clicking on the notification button will allow the user to see notifications about new events, what events your friends are going to, and events that are about to happen.

PROFILE PAGE-1

This is the User's profile where it holds 3 distinct tabs. The first tab shows your upcoming events in chronological order.

PROFILE PAGE-2

The second tab shows your list of friends.

PROFILE PAGE-3

The third tab shows a detailed feed of the events your friends are attending.
